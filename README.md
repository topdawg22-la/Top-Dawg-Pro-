# Top Dawg Pro

Top Dawg Pro is a hybrid diagnostic and maintenance management platform 
built for:

• Automotive diagnostic testing and 
programming systems that will have
a 3D platform that could auto learn,
Year Make and Models. 
gasoline and diesel.
• Work order management
• Preventative maintenance tracking
• Cost tracking and asset management

---

## Architecture

Frontend:
- React Native (Expo)
- TypeScript

Backend:
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Cloud Functions (future expansion)

---

## Core Modules

1. Automotive Diagnostics
2. Automotive Programming Systems
3. Work Orders
4. Maintenance Logs
5. Expense Tracking
6. Parts Inventory
7. Live Data Reports
8. Bi-Directional 
9. 3D & 2D Platform
10. Automatic Indicators of 
(sensors DTC ECU live graphs 
and models).
